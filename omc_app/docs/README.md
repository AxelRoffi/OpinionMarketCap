# OpinionMarketCap Documentation

This is the documentation site for OpinionMarketCap, built with Next.js 15 and MDX.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

The docs site will be available at `http://localhost:3003`.

## Features

- **MDX Support**: Write documentation in Markdown with React components
- **Dark Theme**: Consistent with the main platform design
- **Navigation Sidebar**: Easy navigation between different sections
- **Responsive Design**: Works on all devices
- **Search**: Built-in search functionality (coming soon)

## Adding Documentation

1. Create new MDX files in `src/app/`
2. Add navigation links in `src/components/DocsNavigation.tsx`
3. Follow the existing structure and styling

## Deployment

This site will be deployed to `docs.opinionmarketcap.xyz` using Vercel.

## Structure

- `/getting-started` - Platform basics and quick start guide
- `/smart-contracts` - Technical contract documentation
- `/api-reference` - API endpoints and integration guides
- `/trading-guide` - Advanced trading strategies
- `/security` - Security practices and audit reports
- `/integrations` - Third-party integrations and tools
