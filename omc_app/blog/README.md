# OpinionMarketCap Blog

This is the blog site for OpinionMarketCap, built with Next.js 15 and MDX.

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

The blog site will be available at `http://localhost:3004`.

## Features

- **MDX Support**: Write blog posts in Markdown with React components
- **Dark Theme**: Consistent with the main platform design
- **Blog Layout**: Clean, readable blog post layout
- **Categories**: Organize posts by categories
- **RSS Feed**: Auto-generated RSS feed (coming soon)
- **Newsletter**: Email subscription (integration needed)

## Adding Blog Posts

1. Create new MDX files in `src/app/posts/[slug]/`
2. Add frontmatter with metadata (title, date, author, etc.)
3. Write content in Markdown
4. Update the posts list in the main page

## Content Categories

- **Announcements**: Platform updates and releases
- **Technical**: Development insights and technical deep-dives  
- **Tutorials**: How-to guides and educational content
- **Updates**: Regular progress updates and news
- **Community**: Community highlights and events
